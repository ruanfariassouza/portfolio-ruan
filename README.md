# Ruan Farias — Portfolio

Portfólio em React + Vite para apresentar projetos, serviços e posicionamento profissional.

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy na Vercel

Framework: Vite
Build Command: `npm run build`
Output Directory: `dist`
Install Command: `npm install`
